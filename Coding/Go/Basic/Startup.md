# Install

## Mac

```shell
$ brew install go
```

# Hello World

```go
package main

import "fmt"

func main() {
  fmt.Println("Hello World!")
}
```

**运行**

```shell
$ go run hello.go
Hello World!
```

**编译**

```shell
$ go build hello.go
```
