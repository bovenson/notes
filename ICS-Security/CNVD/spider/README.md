# 环境及依赖
- Python3

- requests

  ```shell
  # 安装
  pip3 install requests
  ```

- BeautifulSoup4

  ```shell
  # 安装
  pip3 install beautifulsoup4
  ```

- selenium

  ```shell
  # 安装
  sudo pip3 install selenium
  ```

- geckodriver

  ```shell
  # 下载
  https://github.com/mozilla/geckodriver/releases/

  # 解压
  # 在PATH中可以找到
  # linux/mac 放到 /usr/local/bin
  ```

- pymysql

  ```shell
  pip3 install pymysql
  ```
