# Class Define

```java
class ClassName {

```

# Setting

## in file

### hide empty members

```shell
hide empty members
```

### set scale

```shell
scale 1.5
```

## in commend line

### reset PLANTUML_LIMIT_SIZE

```shell
java -DPLANTUML_LIMIT_SIZE=8192 -jar /path/to/plantuml.jar ...
```
