# 使用插件

- [Bootstrap](https://getbootstrap.com/docs/4.1/getting-started/webpack/)

  ```shell
  $ npm install --save jquery popper.js
  $ npm install --save bootstrap
  ```

  **入口文件添加**

  ```javascript
  import 'bootstrap'
  import 'bootstrap/dist/css/bootstrap.min.css'
  ```
