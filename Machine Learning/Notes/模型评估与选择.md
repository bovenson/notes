# 经验误差与过拟合

## 基本概念

- **错误率**：把分类错误的样本数占样本总数的比例称为“错误率”
- **精度**：精度 = 1 - 错误率
- **误差**：学习器的实际预测输出与样本的真实输出之间的差异称为“误差”
- **训练误差或经验误差**：学习器在训练集上的误差
- **泛化误差**：在新样本上的误差
- **过拟合**：把训练样本自身的一些特点当作了所有潜在样本都会具有的一般性质，这样会导致泛化性能下降
- **欠拟合**：指对训练样本的一般性质尚未学好
