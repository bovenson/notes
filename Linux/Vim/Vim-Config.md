# 配置选项

```shell
# 设置行号
set number

# 语法高亮
syntax on

# 鼠标
## 可用
set mouse=a

# tab
set tabstop=4
set shiftwidth=4
set expandtab
set smarttab

# 对齐与缩进
## 自动对齐
set autoindent
## 智能缩进
set smartindent

# 显示匹配
set showmatch

# 
```
