# 概述

cat - concatenate files and print on the standard output.

Concatenate FILE(s) to standard output.

# 用法

`cat [OPTION] ... [FILE] ...`

# 选项

| 选项   | 说明                 | 示例   |
| ---- | ------------------ | ---- |
|      |                    |      |
| `-b` | 只对非空行显示行号，覆盖`-n`选项 |      |
| `-n` | 显示所有输出行行号          |      |
